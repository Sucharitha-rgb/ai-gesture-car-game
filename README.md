🚗 Gesture Controlled Racing Game

📌 Overview
This is an AI-based racing game controlled using hand gestures instead of a keyboard.

🧠 Technologies Used
- Python
- OpenCV
- MediaPipe
- Pygame

 🎮 Features
- Control car using hand gestures
- Real-time detection
- No keyboard required

 ▶️ How to Run
pip install -r requirements.txt  
python main.py
